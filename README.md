# LEDIJAZZ — сайт бренда

Премиальный сайт-витрина бренда женской одежды **LEDIJAZZ** («платье-флюид», размеры 50–64).
Статический сайт без сборки — достаточно загрузить файлы на любой хостинг.

## Структура

```
index.html         — главная страница (one-page)
collection.html    — лукбук коллекций (линейки)
offer.html         — Публичная оферта (шаблон РФ)
privacy.html       — Политика конфиденциальности (152-ФЗ)
delivery.html      — Доставка и возврат (ЗоЗПП, ПП РФ №2463)
assets/
  css/styles.css   — дизайн-система и стили
  js/config.js     — ★ ВСЕ ССЫЛКИ И РЕКВИЗИТЫ В ОДНОМ МЕСТЕ
  js/main.js       — интерактив (меню, анимации, форма, cookie)
  fonts/           — самохостинг шрифтов (Cormorant Garamond + Manrope, с кириллицей)
  img/
    ledijazz-logo.svg  — фирменный знак (золотая эмблема)
    qr-playlist.png    — QR-код плейлиста (перегенерируйте под свою ссылку)
    grain.png          — текстура зерна (оформление)
```

## Что заменить перед публикацией (5 минут)

Откройте **`assets/js/config.js`** — это единая точка настройки:

1. **`yandexPlaylistUrl`** — ссылка на ваш плейлист Яндекс.Музыки (для кнопок и QR).
2. **`yandexEmbedUrl`** — embed-ссылка плеера. На Яндекс.Музыке: «Поделиться → Вставить»,
   скопируйте `src` (вид `https://music.yandex.ru/iframe/playlist/USER/КОД`).
3. **`links`** — ссылки на Wildberries, Ozon, Lamoda, Telegram, VK.
4. **`contacts`** — e-mail и телефон.
5. **`requisites`** — реквизиты ИП/ООО (ИНН, ОГРН/ОГРНИП, адрес).
   Эти значения автоматически подставятся в подвал и юридические страницы.

Все плейсхолдеры на сайте помечены, пока данные не заданы.

### QR-код под вашу ссылку
После указания `yandexPlaylistUrl` перегенерируйте картинку QR:
```bash
pip install qrcode pillow
python3 - <<'PY'
import qrcode
img = qrcode.make("ВАША_ССЫЛКА_ПЛЕЙЛИСТА")
img.save("assets/img/qr-playlist.png")
PY
```

## Как добавить свои фотографии

Места под фото оформлены как элементы `.editorial` (с красивой заглушкой).
Чтобы вставить реальное фото — задайте переменную `--img`:

```html
<div class="editorial" style="--img:url('assets/img/look-1.jpg')">…</div>
```

Заглушка (эмблема + подпись) скроется автоматически. Рекомендуемый размер фото:
ширина 1200–1600 px, формат `.jpg`/`.webp`, вес до ~300 КБ (оптимизируйте перед загрузкой).

> Примечание: исходные фото из вашего Google Drive (8–34 МБ каждое) слишком тяжёлые для веба —
> экспортируйте уменьшенные версии и положите в `assets/img/`.

## Юридическая часть (важно)

Страницы `offer.html`, `privacy.html`, `delivery.html` — это **шаблоны** по законам РФ
(152-ФЗ, Закон «О защите прав потребителей», ПП РФ №2463). Перед публикацией:
- подставьте реальные реквизиты;
- **согласуйте тексты с юристом** под вашу модель продаж;
- при отдельном интернет-магазине с онлайн-оплатой учтите требования 54-ФЗ (онлайн-касса).
